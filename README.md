## Features
- SSID (network name) input  
- Security type: `WPA` / `WEP` / `Open (nopass)`  
- Hidden SSID toggle  
- Browser-based QR generation (via QRCode.js CDN)  
- Live card preview  
- Copy Wi-Fi string button  
- Download card as PNG  
- Print / Save as PDF
